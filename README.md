# 🚀 HeavyUser App Support Center

**HeavyUser**는 일상의 불편함을 해결하고 소소한 즐거움을 더하는 다양한 범용 유틸리티 프로그램을 개발합니다. 
이 페이지는 **왓이트(What Eat)**, **커피내기(Coffee Bet)** 등 저희가 제공하는 서비스에 대한 지원 정보를 담고 있습니다.

---

## 📱 서비스 라인업 (Service Lineup)

### 🥗 왓이트 (What Eat)
사용자의 건강 상태에 맞춘 스마트 식단 추천 비서입니다.
* **AI 음식 분석:** Google Gemini AI를 통해 사진 속 음식을 인식합니다.
* **맞춤형 가이드:** 당뇨, 고혈압 등 선택한 건강 상태에 적합한 음식을 제안합니다.
* **식사 기록:** 추천받은 식단을 기기 내 로컬 DB에 안전하게 기록하고 관리합니다.

### ☕ 커피내기 (Coffee Bet)
모임이나 사무실에서 '누가 커피를 살 것인가?'라는 인류 최대의 고민을 해결해 드립니다.
* **공정하고 즐거운 선정:** 직관적인 인터페이스로 누구나 쉽고 빠르게 당첨자를 뽑습니다.
* **다양한 상황 활용:** 점심 내기, 간식 쏘기 등 소규모 모임의 의사결정을 돕는 유틸리티입니다.

## 🛠 주요 기능 및 권한 안내

### [공통 권한 요구사항]
앱의 원활한 기능 제공을 위해 다음과 같은 접근 권한을 요청할 수 있습니다:
* **카메라 (Camera):** 음식 촬영(왓이트) 및 서비스 내 이미지 획득을 위해 필요합니다.
* **사진 라이브러리 (Gallery):** 기기에 저장된 사진을 불러와 분석하기 위해 사용됩니다.

## 🔒 개인정보 처리방침 (Privacy Policy)

HeavyUser는 사용자의 데이터를 최우선으로 보호하며, 불필요한 개인 식별 정보를 수집하지 않습니다.

1. **데이터 저장:** 사용자의 모든 식사 기록 및 게임 결과는 **사용자의 기기 내부(Local DB)에만 저장**됩니다. 앱 삭제 시 데이터는 즉시 소멸됩니다.
2. **AI 분석 데이터:** '왓이트'의 음식 분석을 위해 전송되는 이미지는 Google Gemini API를 통해서만 처리되며, 분석 목적 외에는 절대 저장되거나 공유되지 않습니다.
3. **제3자 제공:** 원칙적으로 사용자의 동의 없이 개인정보를 외부에 제공하지 않습니다. (단, 분석을 위한 API 연동 제외)
